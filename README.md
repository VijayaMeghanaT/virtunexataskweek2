# ASCII Value Finder

## Features:
- GUI-based input using Swing
- Input validation
- ASCII conversion
- (Optional) SQLite logging

## How to Run:
1. Compile: `javac MainApp.java`
2. Run: `java MainApp`
3. JAR: `jar cfe ASCIIApp.jar MainApp *.class`

## Dependencies:
- Java 17+
- (Optional) SQLite JDBC driver
